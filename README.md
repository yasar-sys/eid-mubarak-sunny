# 🌙 Cinematic Eid Mubarak Experience

A premium, interactive, and cinematic web experience designed to share the joy of Eid-ul-Fitr. This project features a 5-scene narrative journey with advanced animations, spiritual Islamic aesthetics, and a personalized card generator.

## ✨ Features

- **Personalized Setup**: Start your journey by entering your details (Name, Gender, and personalized Sender/Receiver names).
- **Realistic Interactive Envelope**: A high-fidelity, texture-aware 3D envelope with a dripping wax seal and realistic fold physics.
- **Cinematic Narrative**: A multi-scene experience featuring spiritual transitions, including a crescent moon scene, lanterns, and mosque silhouettes.
- **Customizable Card Templates**: Choose from three premium card designs:
  - **Royal Teal**: Deep teal with radiant gold foil.
  - **Majestic Midnight**: Indigo blues with shimmering silver accents.
  - **Eternal Ivory**: Warm cream with elegant rose gold finishes.
- **Micro-interactions**: Parallax effects, mouse-tilt responsiveness, and smooth spring-based animations.
- **Audio Integration**: Spiritual ambient background music and interactive sound effects (toggleable).

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Effects**: `canvas-confetti`
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16+)
- [npm](https://www.npmjs.com/) or [bun](https://bun.sh/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yasar-sys/eid-mubarak-sunny.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

```bash
npm run build
```
