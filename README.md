# 🌙 Cinematic Eid Mubarak Experience

A world-class, interactive, and cinematic web odyssey designed to celebrate Eid-ul-Fitr. This project transcends traditional digital greetings by combining high-fidelity 3D interaction, personalized storytelling, and a premium Islamic aesthetic.

---

## 🏆 Award-Winning Interactive Features

### 1. Immersive 5-Scene Storytelling
The experience is not just a page; it's a **journey**. Users are guided through a carefully choreographed 5-scene narrative that builds emotional resonance before the final reveal.
- **Cinematic Transitions**: Seamless scene-switching with high-end blur, scale, and opacity easing.
- **Atmospheric Backgrounds**: Dynamic, cross-fading photographic slideshows of spiritual Eid celebrations, creating a deep sense of presence.

### 2. Hyper-Realistic Physical Interaction
The heart of the experience is the **Interactive Envelope**, engineered with physical realism in mind.
- **Advanced Z-Stacking**: A complex 4-layer composition (Back - Card - Front - Flap) that ensures the card is truly "contained" within the envelope.
- **3D Flap Physics**: A 180-degree 3D rotation (`rotateX`) that mirrors the physical act of opening an invitation.
- **Synchronized Reveal**: The card's visibility and scale are calibrated to animate only after the seal is broken and the flap starts its ascent.

### 3. Personalization & User Identity
The app offers a premium **Identity Setup** phase that goes beyond simple text input.
- **Gender-Adaptive Greetings**: The system intelligently adjusts its vocabulary (e.g., "Brother", "Sister", "Dear") based on user input for a deeply personal touch.
- **Real-time Theming**: Choose between **Royal Teal**, **Majestic Midnight**, or **Eternal Ivory** templates, each with its own unique gradient and accent profile.

### 4. Visual Excellence & Micro-Interactions
- **Golden Magic Trail**: A reactive mouse-trailing particle system that makes every movement feel magical.
- **Dynamic Typographic Reveal**: A typing effect for the final message that synchronizes with celebratory confetti and ambient glow pulses.
- **Parallax Star Fields**: Multi-layered background particles and shooting stars that react to the user's cursor position.

## 🛠️ Tech Stack

- **Foundational Core**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animation Engine**: [Framer Motion](https://www.framer.com/motion/) (Advanced AnimatePresence and Spring Physics)
- **Styling Architecture**: [Tailwind CSS](https://tailwindcss.com/)
- **Visual Effects**: `canvas-confetti` and custom SVG-masking.
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Execution

1. **Setup**: `npm install`
2. **Launch**: `npm run dev`
3. **Build**: `npm run build`
